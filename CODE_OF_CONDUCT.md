# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, critique work not people, and keep
discussions technical. Harassment, personal attacks, and bad-faith participation
are not tolerated.

Report violations by opening a private security advisory or contacting the
maintainer listed on the repository profile. Reports are handled confidentially.
